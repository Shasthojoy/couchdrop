Upload

Uploading files to Couchdrop is done using the SCP command included with almost every variant of Unix/Linux and sometimes Windows.

#> scp filename username@ssh.couchdrop.io:

You can use your standard CouchDrop.io account or create temporary credentials for added security. These credentials can be shared with friends or customers or be used on insecure servers with no risk.
